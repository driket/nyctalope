# Grid

Display items in a grid layout.

<DocGen componentMetadata={props.componentMetadata} />

## Simple Grid

```jsx
<Grid numCol={5}>
  {Array(10)
    .fill('')
    .map((person) => (
      <PeoplePhoto />
    ))}
</Grid>
```

## Responsive Grid

Columns number can be adjusted depending on the screeen width (responsive)

```jsx
<Grid numCol={5} numColXS={1} numColSM={1} numColMD={2} numColLG={4}>
  {Array(10)
    .fill('')
    .map((person) => (
      <PeoplePhoto />
    ))}
</Grid>
```
